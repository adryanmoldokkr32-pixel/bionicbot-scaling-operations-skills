# Viral Flywheel Engineering

This skill focuses on the 'Amazon/Uber' model where each new user or action makes the system stronger and attracts even more users.

## Instructions
1. Identify the 'Core Loop': User Action -> Value Created -> Network Strength -> New User.
2. Design incentives for users to invite others (Referral 2.0).
3. Implement 'Data flywheels' where more data leads to better AI, which leads to more users.
4. Optimize friction points to ensure the wheel spins faster with every cycle.

## Examples
- "Design a flywheel for BionicSovereign where every trade alert shared brings 5 new users."
