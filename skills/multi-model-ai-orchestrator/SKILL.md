# Multi-Model AI Orchestrator

This skill focuses on knowing which AI (Claude, GPT-4, Gemini, Midjourney) is best for each specific task in a project.

## Instructions

1. Use Claude for complex reasoning and long-context coding.
2. Use GPT-4 for creative writing and standard logic.
3. Use Gemini for deep web research and high-volume data analysis.
4. Use Midjourney/DALL-E for high-end artistic visuals.
5. Synthesize outputs from multiple models into one cohesive final project.

## Examples

- "Draft the code in Claude, but use GPT-4 to write the marketing copy for it."
- "Research the market with Gemini, then have Claude plan the software architecture."
