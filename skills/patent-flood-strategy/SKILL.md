# Patent Flood Strategy

This skill focuses on creating a 'Legal Fortress' by filing many micro-patents that together block any competitor from entering our space.

## Instructions
1. Continuously extract 'Novel Logic' from our daily code commits.
2. Draft automated patent applications for each small technical breakthrough.
3. Create a 'Defensive Patent Portfolio' to deter larger giants from litigation.
4. Monitor competitors' filings to identify 'Blocking' opportunities.

## Examples
- "Find 10 patentable steps in our 8K-to-Social rendering pipeline and draft filings."
