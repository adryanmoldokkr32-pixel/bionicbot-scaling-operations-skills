# Automated Customer Success

This skill focuses on 'Scalable Happiness' - ensuring every user gets results from your product automatically.

## Instructions

1. Design interactive 'Product Tours' for new users.
2. Set up an automated 'Knowledge Base' or FAQ bot.
3. Implement 'Success Milestones' (e.g., automated email when a user hits a goal).
4. Create automated surveys (NPS) to track customer satisfaction.
5. Use AI to draft replies to common support tickets automatically.

## Examples

- "Draft an automated email for a user who just completed their first project in the app."
- "Create a structure for a 'Self-Service' help center for my new SaaS."
