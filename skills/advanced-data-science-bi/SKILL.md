# Advanced Data Science & BI

This skill focuses on 'Data Intelligence', using statistics and machine learning to find hidden business opportunities.

## Instructions

1. Perform exploratory data analysis (EDA) on large CSV/SQL datasets.
2. Create predictive models for sales, churn, or market trends.
3. Use clustering to segment customers into 'Personas'.
4. Visualize complex data using advanced charts (Heatmaps, Sankey diagrams).
5. Identify statistically significant correlations between different variables.

## Examples

- "Analyze this year's sales data: which customer segment is most profitable?"
- "Predict next month's revenue based on current lead volume and conversion rate."
