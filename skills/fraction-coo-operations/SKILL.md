# Fractional COO (Operations)

This skill focuses on the 'Machine' inside the business, ensuring everything runs with 100% efficiency.

## Instructions

1. Document 'Standard Operating Procedures' (SOPs) for every repetitive task.
2. Audit current tool stack for redundancy and cost (SaaS audit).
3. Implement project management frameworks (Agile, Kanban, EOS).
4. Automate data entry between different departments.
5. Design onboarding flows for new team members.

## Examples

- "Create an SOP for our weekly content creation process."
- "Suggest a better way to coordinate tasks between the Dev and Marketing teams."
