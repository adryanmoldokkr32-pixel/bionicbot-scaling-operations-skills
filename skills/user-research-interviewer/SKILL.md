# User Research & Interviewer

This skill focuses on 'The Voice of the Customer', ensuring you build what people actually want to buy.

## Instructions

1. Draft open-ended questions for user interviews (The 'Mom Test' style).
2. Analyze call transcripts for recurring 'Pain Points' and 'Feature Requests'.
3. Set up automated survey flows after key user actions.
4. Use sentiment analysis to gauge reaction to new features.
5. Propose product changes based on data, not guesses.

## Examples

- "Analyze these 5 user feedback forms: what is the #1 feature they are asking for?"
- "Draft an interview script to find out why people are cancelling their trial."
